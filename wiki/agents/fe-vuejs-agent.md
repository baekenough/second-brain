# fe-vuejs-agent

Vue 3 developer for Composition API, single-file components, Pinia state management, and Vue Router navigation.

## Overview

`fe-vuejs-agent` builds Vue 3 applications using the Composition API and `<script setup>` syntax — Vue 3's modern development model that replaces the Options API for complex components. It covers the standard Vue ecosystem: Pinia for state (replacing Vuex), Vue Router for navigation, and Vite as the build tool.

## Key Details

- **Model**: sonnet | **Effort**: medium | **Memory**: project
- **Domain**: frontend | **Skills**: impeccable-design, web-design-guidelines

### Capabilities

- Composition API: `ref`, `reactive`, `computed`, `watch`, composables
- `<script setup>` syntax for concise component definition
- Single-file components (`.vue`) with scoped CSS
- Pinia stores for type-safe, devtools-friendly state
- Vue Router: dynamic routes, navigation guards, lazy loading
- Vite configuration for optimal build performance
