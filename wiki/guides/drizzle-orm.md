# Guide: Drizzle ORM

Reference documentation for Drizzle ORM — TypeScript-first ORM for relational databases with type-safe query builder.

## Overview

The Drizzle ORM guide provides reference documentation for TypeScript database access using Drizzle — a lightweight, type-safe ORM that generates SQL-first queries without N+1 query problems. It covers schema definition, migrations, query patterns, and PostgreSQL-specific features.

## Key Topics

- **Schema Definition**: TypeScript-first schema with inferred types
- **Migrations**: `drizzle-kit` migration generation and management
- **Query Patterns**: Type-safe select, insert, update, delete with joins
- **Relations**: One-to-one, one-to-many, many-to-many relation definitions
- **PostgreSQL Features**: JSON, arrays, ENUM types, custom columns
- **Transactions**: Atomic operations and nested transactions
